## 规则：
    红方与黑方
    红方先手
    轮流单击格子
    只能单击己方颜色的格子或是空白的格子
    若单击的是空白格子则该将格子涂上己方颜色
    若单击的是己方颜色的格子则：
        此格子变成空白
        并且相邻的格子中：
            空白涂上己方颜色
            对方颜色的格子和己方颜色的格子变成空白
    不能单击对方颜色的格子
